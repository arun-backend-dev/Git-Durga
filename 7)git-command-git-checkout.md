git checkout:
--------------
git checkout -- <file> is a command that forcibly discards all uncommitted changes in a file by replacing its current content with the exact version stored in the last commit (HEAD).


1)git checkout
---------------
-it shows all the files which are modified and are eligible for checkout

2)git checkout -- file1.txt
----------------------------
-it replaces the file1.txt of working directory with file1.txt of staging area 

3)git checkout .
------------------
-it replaces all files which are eligible for checkout i mean all files which are eligible
